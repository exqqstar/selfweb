# 学习笔记

从 React Native 开发者到 Web 全栈的学习路径。
每个分类下的文件按编号排列，**建议按顺序学习**。

## 学习路线图

```
01-web-basics     Web 基础（先补这些）
     ↓
02-react-web      React 在 Web 端的差异（RN → Web 过渡）
     ↓
03-nextjs         Next.js 框架（项目核心）
     ↓
04-styling        样式系统（Tailwind + CSS）
     ↓
05-devops         部署运维（Docker + VPS）
```

## 分类索引

| 目录 | 内容 | 你的基础 |
|------|------|---------|
| [01-web-basics](./01-web-basics/) | HTML 语义化、CSS 布局、浏览器基础 | 需要补 |
| [02-react-web](./02-react-web/) | RN 与 Web 差异、DOM、事件系统 | 需要过渡 |
| [03-nextjs](./03-nextjs/) | App Router、Server Components、数据获取 | 从零学 |
| [04-styling](./04-styling/) | Tailwind、响应式设计、动画 | 从零学 |
| [05-devops](./05-devops/) | Docker、Caddy、CI/CD | 有基础可跳 |

## 怎么使用

1. 按分类编号顺序推进
2. 每个分类下的文件也有编号，按顺序读
3. 每个笔记文件有 `## 我的实践` 区域 — 写下你自己的理解和代码
4. 标记 ✅ 表示已掌握，🔄 表示还需要复习
