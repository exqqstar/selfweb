# 01 - Web 基础

> RN 帮你隐藏了很多 Web 底层概念，这里补回来。

## 学习顺序

1. [01-html-semantics.md](./01-html-semantics.md) — HTML 语义化标签（SEO 的基础）
2. [02-css-layout.md](./02-css-layout.md) — CSS 布局模型（和 RN 的关键差异）
3. [03-browser-basics.md](./03-browser-basics.md) — 浏览器工作原理（渲染、缓存、DevTools）

## 为什么要学这些

在 RN 里你写 `<View>` 和 StyleSheet，框架帮你处理了底层渲染。
Web 开发中你直接面对 HTML/CSS/浏览器，理解这些基础会让你知道
Next.js 和 Tailwind 在帮你做什么，出问题时也知道往哪里查。
